# Advent of Code exercises
[Advent of code <3](https://adventofcode.com/)

Build and run all solutions from the root advent directory.

#### Building go, cpp or zig solutions

>```
>./build [lang] [year] [day]
>```
>examples
>```
>./build go 2024 21
>./build zig 2024 1
>```

---

#### Running go, cpp or zig solutions

>```
>./run [lang] [year] [day]
>```
>examples
>```
>./run go 2024 21
>./run cpp 2024 1
>```
